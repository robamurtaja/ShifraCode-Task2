# ShifraCode-Task2
# شيفرة كود - موقع تدريب وخدمات

موقع Front-End لشركة **شيفرة كود** يقدم تجربة متكاملة لعرض الدورات، نظام الحجز، معرض الأعمال وصفحة التواصل. تم تصميم الموقع باستخدام **HTML / CSS / JavaScript** ويتميز بالـ **Responsive Design** ليعمل على الأجهزة المختلفة.

---
## الصفحات والميزات

### 1. الصفحة الرئيسية
- صورة Hero أو تعريفية بسيطة.
- نص تعريفي عن الشركة.
- زر **احجز دورة** يفتح نموذج الحجز مباشرة.

### 2. صفحة الدورات
- عرض الدورات في بطاقات (Cards).
- كل بطاقة تحتوي على: اسم الدورة، وصف مختصر، المدة، السعر.
- زر **احجز الآن** يفتح نموذج الحجز.

### 3. نموذج الحجز
- حقول: الاسم، رقم الجوال، البريد، اختيار الدورة.
- بعد الإرسال، يظهر **تنبيه نجاح** ويتم حفظ الحجز في **localStorage**.

### 4. معرض الأعمال (Portfolio)
- شبكة صور (Grid) لأعمال الشركة.
- عند الضغط على أي صورة، تظهر مكبرة داخل **Modal**.

### 5. صفحة تواصل
- نموذج إرسال رسالة.
- روابط الشبكات الاجتماعية: Instagram، Facebook، WhatsApp.
---
## المميزات

- **نظام الحجز:** الحجز يتم حفظه محليًا باستخدام `localStorage`.
- تصميم نظيف ومنظم.
- سهولة التوسع بإضافة دورات أو أعمال جديدة.
